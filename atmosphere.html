Ejecutar CFW (NSW) (Unpatched&IPatched)
Hekate es un bootloader que permite administrar la NAND, emuNANDS, cargar un CFW y mucho más, además todo se ejecuta antes del Home de la Switch (Horizon) y funciona tanto en una Switch unpatched (Bug de RCM) cómo con una Switch Ipatched (parcheada por Nintendo).


Requisitos para ejecutar un CFW:

1.- Una Nintendo Switch UnPatched o IPatched
2.- Descargar este pack de archivos
3.- Seguir las instrucciones


1.- Una Nintendo Switch UnPatched (Bug RCM)

Si cuentas con una Nintendo Switch UnPatched (compatible con el bug de RCM de las Tegra X1 que se comercializaron  hasta junio de 2018), puedes ejecutar el payload con hardware adicional.
Para cargar un payload, debes acceder primero al modo RCM

2.- Una Nintendo Switch IPatched (Parcheada)
Las Nintendo Switch parcheadas pueden ejecutar payloads por software mediante un Warm Boot que ejecuta el explpit, carga el payload y reinicia tu consola con un CFW.

Puedes seguir esta guía para ejecutar Caffeine mediante PegaScape si cuentas con una consola IPatched
